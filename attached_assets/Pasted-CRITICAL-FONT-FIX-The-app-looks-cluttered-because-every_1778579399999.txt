CRITICAL FONT FIX — The app looks cluttered because 
everything is bold. Fix font weights to create proper 
visual hierarchy.

---

## THE CORE PROBLEM

fontWeight: 900 is used everywhere — labels, body text, 
values, descriptions — making everything fight for attention.
Nothing feels elegant because there's no contrast.

---

## NEW FONT WEIGHT SYSTEM

Apply these rules globally across ALL components:

### Anton — display/headlines only
fontFamily: "'Anton', sans-serif"
(Anton has no weight variants — never set fontWeight on Anton)
Use for: page titles, section titles, card headlines, 
button text, nav labels, modal titles, grade letters

### Barlow Condensed 900 — SHORT labels only  
fontFamily: "'Barlow Condensed', sans-serif"
fontWeight: 900
letterSpacing: '1.5px' minimum
Use ONLY for: 2-3 word uppercase labels like 
"HUMIDITY", "TODAY", "COMPLETE", "HIGH", "UV INDEX"
NEVER use for sentences or descriptive text

### Barlow Condensed 400 — body/descriptions
fontFamily: "'Barlow Condensed', sans-serif"
fontWeight: 400  ← THIS IS THE KEY CHANGE
fontStyle: 'normal' OR 'italic'
Use for: all descriptions, subtitles, sentences, 
explanatory text, row values, food names, reasons,
anything more than 3 words that isn't a title

### JetBrains Mono 400 — numbers only
fontFamily: "'JetBrains Mono', monospace"
fontWeight: 400
Use for: ALL numbers

## GLOBAL RULES — apply to ALL other pages:

1. Any text that is a SENTENCE (5+ words) → Barlow 400
2. Any text that is a SHORT LABEL (1-3 words, uppercase) → Barlow 900 tracked
3. Any NUMBER → JetBrains Mono 400
4. Any TITLE/HEADLINE → Anton
5. fontWeight: 900 ONLY on 2-3 word uppercase tracked labels
6. NEVER fontWeight 900 on sentences or paragraphs

## ALSO UPDATE GOOGLE FONTS IMPORT

Make sure Barlow Condensed 400 is imported.
In index.css, update @import:

@import url('https://fonts.googleapis.com/css2?family=Anton&family=Barlow+Condensed:ital,wght@0,400;0,900;1,400;1,900&family=JetBrains+Mono:wght@400;700&display=swap');

This imports all 4 needed variants:
- Barlow Condensed 400 normal
- Barlow Condensed 400 italic  
- Barlow Condensed 900 normal
- Barlow Condensed 900 italic
