## STEP 2: ADD FONT VERIFICATION

Add this temporary debug block at the top of App.jsx,
inside useEffect on mount:

useEffect(() => {
  document.fonts.ready.then(() => {
    const fonts = ['Anton', 'Barlow Condensed', 'JetBrains Mono']
    fonts.forEach(font => {
      const loaded = document.fonts.check(`16px "${font}"`)
      console.log(`Font "${font}" loaded:`, loaded)
    })
  })
}, [])

Check browser console — if any font shows false, 
the import URL is wrong.

## STEP 4: AUDIT AND FIX ALL COMPONENTS

Search every component for incorrect font usage
and fix according to these strict rules:

RULE 1 — Anton (display) ONLY for:
Headlines, page titles, section headers, 
button text, card titles, nav labels, modal titles.
Style: fontFamily: "'Anton', sans-serif"
Note: Anton has only regular weight — never set fontWeight on Anton.

RULE 2 — Barlow Condensed 900 for:
Labels, tags, badges, descriptions, 
body text, sublabels, small caps text.
Style: fontFamily: "'Barlow Condensed', sans-serif", fontWeight: 900

RULE 3 — Barlow Condensed 900 Italic for:
AI-generated text, summaries, insights, quotes, tips.
Style: fontFamily: "'Barlow Condensed', sans-serif", fontWeight: 900, fontStyle: 'italic'

RULE 4 — JetBrains Mono for:
ALL numbers — temperature, AQI, BMI, weight, 
heart rate, progress counters (5/5), 
time displays, calories, water amounts, 
percentages, any numerical data.
Style: fontFamily: "'JetBrains Mono', monospace"

## STEP 5: FIX SPECIFIC VIOLATIONS

Find and fix these common violations:

1. Any element showing a NUMBER but using Barlow Condensed:
   → Change to JetBrains Mono

2. Any HEADLINE or TITLE using Barlow Condensed:
   → Change to Anton

3. Any element with fontFamily containing 'Bebas Neue':
   → Replace with Anton

4. Any element with fontFamily containing 'Orbitron':
   → Replace with JetBrains Mono

Run these checks in terminal:
grep -r "Bebas Neue" src/
grep -r "Orbitron" src/
grep -r "font-family.*Barlow" src/ | grep -i "title\|header\|headline"

All should return empty or be fixed.

## STEP 7: INDEX.CSS GLOBAL RULES

Ensure these global rules exist in index.css:

  box-sizing: border-box;
  margin: 0;
  padding: 0;
}

body {
  font-family: 'Barlow Condensed', sans-serif;
  font-weight: 900;
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
}

/* Override browser defaults for inputs */
input, textarea, select, button {
  font-family: inherit;
}

/* Numbers always JetBrains Mono */
.num {
  font-family: 'JetBrains Mono', monospace;
}

/* Display text always Anton */
.display {
  font-family: 'Anton', sans-serif;
}
