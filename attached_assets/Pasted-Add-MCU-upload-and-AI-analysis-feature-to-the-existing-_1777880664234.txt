BACKEND (add to existing Express server or create server.js):

1. Install dependencies:
   npm install express multer @anthropic-ai/sdk cors dotenv

2. Create/update server.js with these endpoints:

   POST /api/analyze-mcu
   - Accept multipart form data (file upload)
   - Accept: JPG, PNG, PDF, max 10MB
   - Convert file to base64
   - Send to Claude API (claude-opus-4-5) with this prompt:
   
   System prompt:
   "You are a medical data analyst for 20FIT Sport Clinic Indonesia. 
   Analyze this medical checkup (MCU) document and return a JSON response only.
   No markdown, no explanation outside JSON."
   
   User prompt:
   "Analyze this MCU document. Return ONLY valid JSON with this exact structure:
   {
     'summary': 'Brief 2-sentence overall health summary in Bahasa Indonesia',
     'grade': 'A/B/C/D based on overall health',
     'metrics': [
       {'label': 'metric name', 'value': 'value with unit', 'status': 'ok/high/low/warning', 'note': 'brief note'}
     ],
     'recommendations': ['recommendation 1', 'recommendation 2', ...],
     'checklist': [
       {'icon': 'emoji', 'title': 'task title', 'reason': 'why this task', 'priority': 'high/med/low', 'duration': number_or_null, 'location': 'gym/home/clinic/null'},
       ... (generate 5 personalized daily tasks based on the MCU results)
     ],
     'doctor_notes': 'Key medical findings in Bahasa Indonesia',
     'reviewed_at': 'today date'
   }"
   
   - Return the JSON to frontend
   - Handle errors gracefully

3. Add CORS for frontend origin
4. Store ANTHROPIC_API_KEY in Replit Secrets (env var)
5. Server listens on process.env.PORT || 3000
