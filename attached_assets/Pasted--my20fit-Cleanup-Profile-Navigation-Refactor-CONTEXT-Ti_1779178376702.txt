## TASK C — PROFILE ACCESS VIA USER CARD (EMBEDDED)

Ini bagian utama yang baru — pattern Discord/Slack/Spotify style.

### Step C1 — Remove Profile dari Main Menu

```javascript

```

**Hasilnya:** Main menu sidebar tinggal 4 items:
- HOME
- NUTRITION
- PROGRESS
- MOMENTS

### Step C2 — Make User Card di Bottom-Left Clickable

Di `Sidebar.jsx`, cari section user card di bottom-left yang menampilkan:
- Avatar initials (lingkaran merah dengan "ZI")
- Nama "ZIDNI"
- Email "zidni@20fit.id"
- Badge "PLUS MEMBER"

Saat ini kemungkinan struktur-nya:
```jsx
<div className="user-card">
  <div className="avatar">ZI</div>
  <div className="user-info">
    <p className="name">ZIDNI</p>
    <p className="email">zidni@20fit.id</p>
    <span className="badge">⭐ PLUS MEMBER</span>
  </div>
</div>
```

**Transform jadi:**
```jsx
import { useNavigate, useLocation } from 'react-router-dom';

// inside component:
const navigate = useNavigate();
const location = useLocation();
const isProfileActive = location.pathname === '/profile';

<button
  type="button"
  onClick={() => navigate('/profile')}
  className={`user-card-button ${isProfileActive ? 'active' : ''}`}
  aria-label="Buka Profile"
>
  <div className="avatar">ZI</div>
  <div className="user-info">
    <p className="name">ZIDNI</p>
    <p className="email">zidni@20fit.id</p>
    <span className="badge">⭐ PLUS MEMBER</span>
  </div>
</button>
```

### Step C3 — Add Visual Feedback (Hover + Active State)

User card button perlu visual cue supaya jelas clickable:

**Styling guidelines (pakai Tailwind atau inline style sesuai existing pattern):**

```jsx
// Base state — looks like existing card, but clearly tappable
style={{
  background: 'transparent', // atau existing bg
  border: 'none',
  width: '100%',
  textAlign: 'left',
  padding: '12px 16px',
  cursor: 'pointer',
  borderRadius: '12px',
  transition: 'all 0.2s ease',
  display: 'flex',
  alignItems: 'center',
  gap: '12px',
}}

// Hover state
':hover': {
  background: 'rgba(255, 255, 255, 0.05)',
}

// Active state (when /profile route is active)
isProfileActive && {
  background: 'rgba(192, 57, 43, 0.15)', // subtle red tint
  borderLeft: '3px solid #C0392B',       // red accent bar (consistent dengan active menu style)
}
```

**Penting:** Active state styling-nya **konsisten dengan menu items lain** yang aktif. Kalau menu HOME aktif pakai red bar di kiri + bg tint, user card juga harus pakai pattern sama.
