The problem: everything looks the same weight and style.

## THE RULE (apply everywhere, no exceptions):

Anton → ONLY for: page titles, section titles, 
card headlines, button text, nav labels, modal titles,
grade letters, ALL CAPS display text.
Weight: normal (Anton has no bold variant)
Never use on body text, descriptions, or labels.

Barlow Condensed 900 → ONLY for: 
uppercase tracked labels (HUMIDITY, TODAY, COMPLETE, HIGH),
tag/badge/pill text, small caps metadata.
Always: fontWeight: 900, letterSpacing: '1.5px or more'
Never use for long descriptions or body paragraphs.

Barlow Condensed 900 Italic → ONLY for:
body descriptions, subtitles, condition text,
AI-generated summaries, tips, hints, motivational text,
anything that is a sentence or paragraph.
Always: fontWeight: 900, fontStyle: 'italic'

JetBrains Mono → ONLY for:
ALL numbers — temperature, AQI, BMI, weight, time,
progress counters, percentages, calories, distances.
Never use on words.
