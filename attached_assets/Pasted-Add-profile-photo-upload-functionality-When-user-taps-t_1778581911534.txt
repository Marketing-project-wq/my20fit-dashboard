## FEATURE: AVATAR PHOTO UPLOAD

### How it works:
1. User clicks avatar circle
2. Hidden file input triggered
3. User selects image (JPG/PNG/WEBP, max 5MB)
4. Image previewed immediately (base64)
5. Saved to localStorage 'my20fit_avatar'
6. Avatar displays photo instead of initials

---

## IMPLEMENTATION IN Profile.jsx:

### State:

const avatarInputRef = useRef(null)

### File handler:
function handleAvatarChange(e) {
  const file = e.target.files[0]
  if (!file) return

  // Validate type
  const allowed = ['image/jpeg','image/png','image/webp']
  if (!allowed.includes(file.type)) {
    showToast('Format tidak didukung. Gunakan JPG, PNG, atau WEBP.', 'error')
    return
  }

  // Validate size (5MB)
  if (file.size > 5 * 1024 * 1024) {
    showToast('Ukuran foto maksimal 5MB.', 'error')
    return
  }

  // Convert to base64 and save
  const reader = new FileReader()
  reader.onload = (ev) => {
    const base64 = ev.target.result
    setAvatarSrc(base64)
    localStorage.setItem('my20fit_avatar', base64)
    showToast('Foto profil berhasil diperbarui ✓')
  }
  reader.readAsDataURL(file)

  // Reset input so same file can be re-selected
  e.target.value = ''
}

function handleAvatarClick() {
  avatarInputRef.current?.click()
}

### Hidden file input (place anywhere in JSX):

### Also update Sidebar.jsx:
Load avatar from localStorage on mount:
const [avatarSrc, setAvatarSrc] = useState(null)

useEffect(() => {
  const saved = localStorage.getItem('my20fit_avatar')
  if (saved) setAvatarSrc(saved)

  // Listen for storage changes (when profile updates avatar)
  const handleStorage = () => {
    const updated = localStorage.getItem('my20fit_avatar')
    setAvatarSrc(updated)
  }
  window.addEventListener('storage', handleStorage)
  return () => window.removeEventListener('storage', handleStorage)
}, [])

// Also listen via custom event for same-tab updates:
useEffect(() => {
  const handleAvatarUpdate = () => {
    const saved = localStorage.getItem('my20fit_avatar')
    setAvatarSrc(saved)
  }
  window.addEventListener('my20fit_avatar_updated', handleAvatarUpdate)
  return () => window.removeEventListener('my20fit_avatar_updated', handleAvatarUpdate)
}, [])

In Profile.jsx, after saving avatar, dispatch event:
